# Court Hearing Types

## Hearing Types

| Code | Description |
|------|------------|
| Regular | साधारण पेशी — Regular hearing |
| Priority | प्राथमिकता पेशी — Priority hearing |
| Urgent | अति जरुरी — Urgent hearing |
| Mention | उल्लेख — Mention/note |

## Decision Types

| Type | Description |
|------|------------|
| `hearing_held` | Hearing was conducted |
| `adjourned` | मुल्तवी — Postponed |
| `verdict_reserved` | फैसला सुनाउन बाँकी — Judgement reserved |
| `verdict_delivered` | फैसला भएको — Judgement delivered |
| `dismissed` | खारेज — Dismissed |
| `withdrawn` | फिर्ता — Withdrawn |
| `settled` | मिलापत्र — Settled |

## Bench Types

- **Single Bench (एकल इजलास)**: One judge
- **Division Bench (संयुक्त इजलास)**: Two judges
- **Full Bench (पूर्ण इजलास)**: Three or more judges
- **Constitutional Bench (संवैधानिक इजलास)**: Five judges (Supreme Court only, constitutional cases)
