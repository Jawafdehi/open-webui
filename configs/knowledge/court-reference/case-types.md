# NGM Case Type Codes

Used when searching or analyzing court cases.

## Common Case Types

| Code | Meaning |
|------|--------|
| CR | Criminal Case |
| CV | Civil Case |
| WP | Writ Petition |
| RP | Review Petition |
| CP | Contempt Petition |

## Special Court Case Categories

Corruption cases at the Special Court typically involve:

- **CIAA Filed Cases**: Cases filed by the Commission for the Investigation of Abuse of Authority
- **Property Attachment**: Asset seizure proceedings related to corruption
- **Bail Hearings**: Pretrial release decisions in corruption cases

## Verdict Types

| Type | Description |
|------|------------|
| Final Verdict (अन्तिम फैसला) | Final decision disposing of the case |
| Interim Order (अन्तरिम आदेश) | Temporary order during pendency |
| Stay Order (यथास्थिति आदेश) | Order maintaining status quo |

## Hearing Decision Types

- **Hearing Held** — Case was heard
- **Adjourned** — Postponed to next date
- **Verdict Reserved** — Judgment to be delivered later
- **Verdict Delivered** — Judgment announced
- **Dismissed** — Case dismissed
- **Withdrawn** — Case withdrawn by petitioner

## Registration Date Format

Court cases use Bikram Sambat (BS) dates for registration. The `convert_date` tool can convert between AD and BS.
