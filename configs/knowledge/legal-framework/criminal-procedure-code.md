# Criminal Procedure Code — Relevant Sections

## Overview

The National Criminal Procedure Code, 2074 (2017) governs criminal proceedings in Nepal. The following sections are particularly relevant for corruption cases.

## Interaction with Special Court

The Special Court Act, 2059 overrides certain Criminal Procedure Code provisions for corruption cases. The Special Court follows an expedited procedure with specific timelines.
